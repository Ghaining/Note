* 在Web应用中，客户端输入不可信是一个基本原则
* 常见的几种Web前端安全问题
1. XSS（跨站脚本攻击）（cross-site scripting）

    -> 伪造会话（基于XSS实现CSRF）
    
    -> 劫持cookie
    
    -> 恶意代码执行

2. CSRF（跨站请求伪造）（cross-site request forgery）

    -> 伪造用户身份操作
    
3. SQL注入
